プライバシーポリシー

最終更新日: 2026年3月27日

本アプリ「BudgetHub」は運営者sssoftwareにより提供されています。 運営者は、本アプリにおける利用者情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 取得する情報

本アプリは、以下の情報を取り扱います。

  • 利用者が登録した収入支出データ（タイトル、カテゴリ、金額、日付、メモ、固定費設定等）
  • 利用者が任意で取り込んだレシート画像および OCR 処理結果
  • 資産管理機能で利用する保有情報、銘柄情報、価格取得結果
  • アプリ設定情報（表示設定、通知設定、テーマ、言語、アプリアイコン、パスコード、生体認証設定等）
  • CSV 出力、バックアップ、復元に関するデータ（利用者が明示操作したファイル入出力）
  • 購入状態に関する情報（広告削除オプションの購入・復元状態）
  • 広告配信時に外部事業者が取得する情報（端末情報、広告識別子、広告表示・操作情報等）
  • お問い合わせ時に利用者が提供する情報（メールアドレス、問い合わせ内容等）

2. 利用目的

  1. 収入支出データの登録・表示・編集・削除等、本アプリ機能の提供のため
  2. レシート読取、CSV 出力、通知、アプリロック等の補助機能の提供のため
  3. 資産管理機能における保有情報表示および価格情報取得のため
  4. 広告表示および広告配信の最適化のため
  5. 有料オプションの購入確認・復元・利用権限管理のため
  6. 不具合対応およびお問い合わせ対応のため

3. 取得方法・保存先

  1. 収入支出データは、利用者の手入力またはレシート読取により取得されます。
  2. レシート画像は、利用者が撮影または読取操作を実行した場合にのみ取得します。
  3. 本アプリデータは原則として端末内に保存され、運営者独自サーバーへ自動送信されません。
  4. CSV 出力やバックアップは利用者の明示操作時のみ作成され、利用者が保存先を選択します。

4. 外部送信・第三者サービス

本アプリは以下の外部サービスを利用し、機能実行時に通信が発生します。

  • Apple Vision / VisionKit（レシート読取）
  • Apple StoreKit / App Store（課金、購入確認、復元）
  • Google AdMob（広告表示）
  • 市場データ提供サービス（価格情報取得）

これらの外部事業者における情報取扱いは、各事業者のポリシーに従います。

参考リンク:
Apple プライバシーポリシー: https://www.apple.com/legal/privacy/jp/
Google プライバシーポリシー: https://policies.google.com/privacy?hl=ja

5. 権限の利用

  • カメラ権限: レシート撮影機能のため
  • Face ID / Touch ID: アプリロック解除機能のため
  • 通知権限: リマインダー通知機能のため

これらの権限は、iOS 設定画面からいつでも変更できます。

6. 第三者提供

運営者は、次の場合を除き、個人情報を第三者に提供しません。

  • 本人の同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要で、本人同意の取得が困難な場合
  • 公衆衛生の向上または児童の健全育成推進のために特に必要で、本人同意の取得が困難な場合
  • 国の機関または地方公共団体等への協力が必要で、本人同意取得により当該事務に支障を及ぼすおそれがある場合

7. 安全管理措置

  • 端末内保存を前提としたデータ管理
  • アプリ内パスコード / 生体認証機能による保護
  • 不要データの削除機能、CSV 出力、バックアップ / 復元機能の提供

ただし、通信および電子保存の性質上、完全な安全性を保証するものではありません。

8. 利用者の権利

利用者は、本アプリ内で登録データの閲覧・修正・削除を行うことができます。 アプリを端末から削除した場合、端末内データは原則として削除されます（利用者が外部保存したバックアップを除く）。

9. 本ポリシーの変更

運営者は、法令改正やサービス内容の変更等に応じて本ポリシーを改定することがあります。 重要な変更がある場合は、本アプリ内または運営者が適切と判断する方法で周知します。
